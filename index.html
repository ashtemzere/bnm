Overview:
It can (RE) online sign applications and bundle them into IPA files to be installed on IOS devices. Supported platform are: Mobile Phone, Windows, Mac System
Prerequisites:
If you want to use it You need to open it in a browser such as Safari chrome
https://app.online-sign.vip
Usage:
1.First open https://app.online-sign.vip
2.IPA file: select the IPA file after packaging
3. p12 certificate: select the generated p12 and add certificate
4.P12 mobileprovision: select the generated mobileprovision file and upload it
5.P12 Password, if there is a certificate password, enter the password without leaving it blank
6.Finally, click start and you are done!
7.can choose Click to install or Click download ipa to configure internal distribution by yourself
